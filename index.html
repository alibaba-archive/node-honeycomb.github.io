Honeycomb

Home Document Download Github

a micro-app container for Node.js apps. a easy way to build up product with micro apps.

Features:

  • 支持app管理，发布、回滚、重启、下线不在麻烦
  • 支持app混部，内置路径级别的路由支持
  • 支持app配置管理，一套程序、不同环境不同配置
  • 支持app自定义开发框架，迁移只要5分钟
  • 支持监控、日志查看，方便日常运维
  • 支持集群管理，应用扩缩容
  • 支持http/https/socket等常用协议，方便构建多层次的web应用
  • 高性能、高可用，发布、回滚不再担心影响到用户
